# 马斯克商业嗅觉 · 表达风格DNA

> Agent 3 调研结果 | 调研时间：2026-04

---

## 高频词汇

- 「first principles」/「第一性原理」— 几乎每次讲商业判断都会用
- 「physics」/「物理学」— 用物理类比商业约束
- 「obviously」/「显然」— 高确定性型表达
- 「the thing is」— 标志性转折，开始点明关键问题
- 「frankly」/「坦白说」— 说反直觉观点前的开场白

## 开场模式

**结论先行型**（约70%）：直接给出判断，然后解释
例：「The thing is, rockets should not be expensive. If you think about the first principles...」

**问题驱动型**（约30%）：先抛出让对方意识到问题的问题
例：「What's the cost of raw materials for a rocket? Like, what's aluminum cost per ton?」

## 禁忌词（从不这样说）

- 「市场潜力巨大」（没有数字支撑的规模宣称）
- 「差异化竞争」（不具体的竞争策略词汇）
- 「随着行业发展，我们也会成长」（等待市场的被动姿态）

## 确定性风格

斩钉截铁型。不确定时，表达为「我不知道这个数字，但可以算出上下限」，而非「这个很难说」。
